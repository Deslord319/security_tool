核心功能优势

HarmonyShield 深度集成 HarmonyOS 系统能力，基于 netFirewall API 实现精细化的网络流量管控，支持 IP 地址、域名、DNS 三种防火墙规则类型，可按入站/出站方向灵活配置，有效阻断恶意流量和未授权网络访问。在外设管控方面，产品全面管控 USB、蓝牙、Wi-Fi、HDC 等外设接口，支持设备黑白名单策略，从源头防止未授权设备接入和敏感数据外泄。审计日志模块实时记录防火墙操作、设备接入、身份认证等安全事件，支持多维度筛选查询和日志导出，为安全事件追溯和合规审计提供完整证据链。身份认证策略支持口令复杂度配置（长度、大小写、数字、特殊字符）和有效期管理，并集成 HarmonyOS 原生 PIN/指纹/人脸认证能力，强化终端身份安全。安全态势总览页面一屏展示防火墙状态、审计事件统计、已连接外设、身份策略等关键指标，配合快捷功能入口让管理员快速掌控全局。产品采用零信任安全架构，提供细粒度的访问控制能力，支持启动密码校验和多种认证方式组合，确保管理工具自身安全可信。

适用场景

企业 IT 管理

企业 IT 管理员可使用 HarmonyShield 统一管理公司移动设备和 2in1 设备的网络访问策略，通过防火墙规则限制员工访问特定网站或服务，防止工作时间访问娱乐网站，保障网络带宽和工作效率。同时通过外设管控禁止私人 U 盘接入，防止公司数据外泄。

数据防泄漏 · 网络管控 · 效率提升

政府与公共部门

政府机关使用 HarmonyShield 对涉密终端进行严格管控。通过身份认证策略强制使用复杂口令并定期更换，通过审计日志记录所有操作行为，满足等保要求。蓝牙和 Wi-Fi 接口默认禁用，仅允许通过有线网络访问内部系统，确保政务数据安全。

等保合规 · 涉密管控 · 行为审计

医疗与金融机构

医院和银行使用 HarmonyShield 保护患者隐私和客户财务数据。防火墙规则限制终端只能访问业务系统 IP，外设管控禁止使用未认证的 USB 设备，身份鉴别确保只有授权人员才能操作敏感系统，审计日志为合规检查提供完整记录。

隐私保护 · 合规审计 · 访问控制

工业制造与研发

制造企业和研发机构使用 HarmonyShield 保护核心技术和生产数据。通过 DNS 规则阻止访问云存储和邮箱服务防止资料外传，通过 USB 白名单仅允许公司配发的加密 U 盘，通过审计日志追踪可疑的数据访问行为，全方位保护知识产权。

知识产权保护 · 数据安全 · 行为追溯

快速入门

1

安装应用

将签名后的 HAP 包安装到 HarmonyOS 设备。支持通过 HDC 命令行安装或 DevEco Studio 直接部署。

# 使用 HDC 安装
hdc install signApp.hap
提示

本应用使用企业级 API，需要使用正确签名的 HAP 包。详细签名流程请参考项目 README.md 文档。

2

进入安全总览

启动应用后默认进入安全总览页面（Dashboard），可以看到设备当前的安全状态，包括：

  • 防火墙开启状态
  • 今日审计事件数量
  • 已连接外设数量
  • 身份策略执行情况

点击各统计卡片可快速跳转到对应功能模块。

3

配置防火墙规则

在左侧导航栏点击「防火墙」进入防火墙管理页面：

  1. 开启防火墙：点击防火墙开关，完成身份认证后启用网络防火墙
  2. 选择规则类型：支持 IP 地址、域名、DNS 三种规则类型
  3. 设置流量方向：根据需要选择入站或出站流量方向
  4. 添加规则：点击「添加规则」按钮，填写规则详情并保存
注意

防火墙规则会立即生效，请谨慎配置。建议先在测试环境验证规则效果。

4

管理外设接口

在左侧导航栏点击「外设管理」进入外设管控页面，可进行：

  • 接口管控：启用/禁用 USB、蓝牙、Wi-Fi、HDC 等接口
  • 设备白名单：添加允许接入的 USB/蓝牙设备
  • 设备黑名单：阻止特定设备接入
  • 连接记录：查看历史设备连接情况
5

查看审计日志

在左侧导航栏点击「日志管理」进入审计日志页面，支持：

  • 按事件类型筛选（防火墙、外设、身份认证等）
  • 按时间范围查询
  • 关键词搜索
  • 日志导出（CSV/JSON 格式）
