防火墙

通过 netFirewall API 管控网络入站/出站流量，支持 IP、域名、DNS 三种规则类型。

规则类型

IP 规则 基于 IP 地址和端口的访问控制
3
域名规则 基于域名的访问控制
1
DNS 规则 基于 DNS 的访问控制
0

流量方向

入站规则 (RULE_IN) 2 条规则
出站规则 (RULE_OUT) 2 条规则
