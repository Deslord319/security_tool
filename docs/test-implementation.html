测试架构概览

本方案采用四层测试架构，从下往上依次为：单元测试 → 服务层测试 → 组件测试 → E2E 测试。每层测试有明确的职责边界和依赖关系。

E2E 测试 页面快照 + 关键流程验证
需真机
组件测试 StatCard / SideBar / Dialog 渲染与交互
需真机
服务层测试 FirewallService / PeripheralService 业务逻辑
Mock API
单元测试 DataModels / 解析函数 / 格式化函数
纯逻辑
✅ 核心原则

测试金字塔：单元测试数量最多、执行最快；E2E 测试数量最少、覆盖关键路径。80% 的测试应该不依赖 UI，确保 UI 修改不影响大部分测试用例。

⚙

单元测试 - 实现概述

单元测试覆盖项目中的纯函数和数据模型，不依赖任何 UI 组件或系统 API。这类测试执行速度快，可在本地直接运行。

服务层测试 - Mock 机制设计

服务层测试需要 Mock 系统 API（如 netFirewall、usbManager），使测试可以在无设备环境下运行。

Mock 设计原则

✔ 接口一致性
Mock 类的方法签名必须与真实 Service 完全一致，确保可替换。
✔ 可控数据
通过 setMockData() 预设测试数据，模拟各种业务场景。
✔ 异常模拟
支持模拟 API 调用失败，测试错误处理逻辑。

组件测试 - 快照测试原理

快照测试的核心思想是：首次运行时生成组件的"基线快照"，后续运行时对比当前输出与基线，检测意外变更。

快照测试流程

1
渲染组件
→
2
序列化结构
→
3
对比基线
→
4
输出结果

快照工具实现

import { fileIo } from '@kit.CoreFileKit' /** * 组件快照测试工具 */ export interface ComponentSnapshot { componentName: string props: Record<string, unknown> timestamp: number structure: string // 序列化的组件结构 } export interface SnapshotDiff { match: boolean differences: string[] } export class SnapshotManager { private baselinePath: string constructor(baselinePath: string) { this.baselinePath = baselinePath } /** * 生成组件快照 */ createSnapshot(componentName: string, props: Record<string, unknown>, structure: string): ComponentSnapshot { return { componentName, props, timestamp: Date.now(), structure } } /** * 保存为基线 */ async saveBaseline(snapshotName: string, snapshot: ComponentSnapshot): Promise<void> { let path = `${this.baselinePath}/${snapshotName}.json` let content = JSON.stringify(snapshot, null, 2) let file = fileIo.openSync(path, fileIo.OpenMode.CREATE | fileIo.OpenMode.WRITE_ONLY) fileIo.writeSync(file.fd, content) fileIo.closeSync(file) } /** * 加载基线 */ async loadBaseline(snapshotName: string): Promise<ComponentSnapshot | null> { let path = `${this.baselinePath}/${snapshotName}.json` try { let stat = fileIo.statSync(path) if (!stat.isFile()) return null let file = fileIo.openSync(path, fileIo.OpenMode.READ_ONLY) let buf = new ArrayBuffer(stat.size) fileIo.readSync(file.fd, buf) fileIo.closeSync(file) let content = String.fromCharCode(...new Uint8Array(buf)) return JSON.parse(content) as ComponentSnapshot } catch { return null } } /** * 对比快照 */ compareSnapshots(current: ComponentSnapshot, baseline: ComponentSnapshot): SnapshotDiff { let differences: string[] = [] if (current.structure !== baseline.structure) { differences.push('Component structure changed') } let currentProps = JSON.stringify(current.props) let baselineProps = JSON.stringify(baseline.props) if (currentProps !== baselineProps) { differences.push('Props changed') } return { match: differences.length === 0, differences } } }
