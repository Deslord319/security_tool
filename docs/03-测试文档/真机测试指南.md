- [ ] 开启开发者模式

- [ ] 授权电脑调试

## 🚀 快速开始

### 方式 1: 使用测试脚本（推荐）

```bash
# 运行一键测试脚本
test-on-device.bat
```

脚本会自动完成：
1. ✅ 检查设备连接
2. ✅ 构建 HAP
3. ✅ 签名 HAP
4. ✅ 安装到设备
5. ✅ 激活企业管理员
6. ✅ 显示下一步指引

### 方式 2: 手动执行

```bash
# 1. 构建
build_hap.bat

# 2. 签名
cd hapsigner
copy ..\entry\build\default\outputs\default\entry-default-unsigned.hap .
2-debug-sign.bat
cd ..

# 3. 安装
hdc install hapsigner/signApp.hap

# 4. 激活企业管理员（可选，仅企业版设备）
hdc shell edm enable-admin -n com.huawei.securitytool -a EnterpriseAdminAbility -t super

# 5. 启动应用
hdc shell am start -n com.huawei.securitytool/.EntryAbility
```

## 📱 测试流程

### 步骤 1: 安装应用

```bash
# 检查设备连接
hdc list targets

# 应该看到类似输出:
# 192.168.1.100:5555    device
# 或
# HWI-AL00            device
```

如果未检测到设备：
```bash
# 使用无线调试
hdc tconn 192.168.1.100:5555

# 或使用 USB 连接
# 确保 USB 调试已开启并已授权
```

### 步骤 2: 验证安装

```bash
# 查看已安装应用
hdc shell bm list

# 查找 com.huawei.securitytool
```

### 步骤 3: 启动应用

```bash
# 启动应用
hdc shell am start -n com.huawei.securitytool/.EntryAbility

# 或使用 HDC 的启动命令
hdc shell aa start -a EntryAbility -b com.huawei.securitytool
```

### 步骤 4: 查看日志

```bash
# 实时查看应用日志
hdc hilog

# 过滤应用日志
hdc hilog | grep securitytool
```

### 步骤 5: 执行测试

1. **打开应用**
   - 在设备上找到 HarmonyShield 图标
   - 点击打开

2. **按照测试清单测试**
   - 打开 `docs/TEST_CHECKLIST.md`
   - 逐项测试并记录结果

3. **填写测试报告**
   - 复制 `docs/test-report-template.md`
   - 填写测试结果
   - 截图保存

## 🔧 常见问题

### Q1: 安装失败（签名错误）

**错误信息**:
```
Install failed: Signature verification failed
```

**解决方案**:
1. 检查签名配置是否正确
2. 重新生成 p7b 文件
3. 重新签名 HAP
4. 确保 `UnsgnedDebugProfileTemplate.json` 中 bundle-name 正确

---

### Q2: 应用闪退

**现象**: 打开应用后立即关闭

**解决方案**:
1. 查看日志：`hdc hilog`
2. 检查权限声明
3. 检查签名配置
4. 重新安装应用

---

### Q3: MDM 功能失败

**错误信息**:
```
Error 9200001: Permission denied - MDM admin not enabled
```

**解决方案**:
1. 激活企业管理员：
   ```bash
   hdc shell edm enable-admin -n com.huawei.securitytool -a EnterpriseAdminAbility -t super
   ```
2. 验证激活状态：
   ```bash
   hdc shell edm list-admins
   ```
3. 如果失败，检查设备是否为企业版

---

### Q4: 设备未检测到

**现象**: `hdc list targets` 无输出

**解决方案**:
1. 检查 USB 连接
2. 检查 USB 调试是否开启
3. 重新授权电脑
4. 重启 HDC 服务：
   ```bash
   hdc kill
   hdc start
   ```
5. 使用无线调试：
   ```bash
   hdc tconn 192.168.1.100:5555
   ```

---

### Q5: 主题切换不生效

**现象**: 点击主题选项后无变化

**解决方案**:
1. 检查 ThemeManager 初始化
2. 查看日志确认主题切换事件
3. 重启应用
4. 清除应用数据后重试
